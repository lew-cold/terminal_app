Regi-Star: README

App execution: 
To execute the app, extract the files to a chosen folder and run Terminal.

1) After loading Terminal, navigate to the directory of the extracted files.
2) On the command line enter: ‘./app’
3) You should now be in Regi-Star!

Teachers mode:

To access the teachers mode please use the default PIN, which is ‘1234’ This can be changed in the back-end as no front-end facility exists.
